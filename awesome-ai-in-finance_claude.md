## Financial AI Taxonomy Matrix
The index organizes financial intelligence resources into specialized execution sectors:
1. **Autonomous Agents**: Frameworks for multi-agent financial R&D, self-improving trading behaviors, and automated analysis (e.g., `FinRobot`, `ATLAS`, `Vibe-Trading`, `OpenFinClaw`).
2. **Financial Large Language Models (LLMs)**: Specialized financial language engines, dataset benchmarks, and fine-tuning pipelines (e.g., `FinGPT`, `PIXIU`, `DeepSeek-R1` financial adaptations, `AI Hedge Fund`).
3. **Advanced Machine Learning Strategies**: Deep Reinforcement Learning (DRL) libraries (`FinRL`), long-horizon time-series forecasting, and distribution modeling (`wasserstein-btc`).
4. **Alternative Data & Alpha Ingestion**: Tools scoring alternative market triggers including global news sentiment, prediction markets (`Polymarket`/`Kalshi` via `oracle3`/`Parsec`), and unusual geopolitical signals (Pentagon Pizza Index).
5. **Modern Model Context Protocol (MCP)**: System bridges allowing AI agents to connect directly to secure brokerage backends (`Trade It`) or pull live technical data streams (`FinanceKit MCP`).
